# Parky – Project Scope (v1)

## Vision
Help drivers find parking faster by showing a parking confidence overlay near a destination.

## In Scope (v1)
- Destination search + map view
- Street-segment based “parking confidence” layer
- User reporting: “I left a spot”
- Confidence decay over time
- Basic trust + rate limiting to reduce spam
- Realtime updates on the map

## Out of Scope (v1)
- Guaranteed real-time spot availability
- City sensor integrations
- Computer vision
- Payments / subscriptions

## Success Criteria
- User can search a destination and see top nearby street segments
- User can report a freed spot in under 10 seconds
- Confidence updates within seconds
- Deployed demo link is available
